## 摘要
生成式建模的最新进展使得生成的表格数据成为隐私敏感数据共享的实用解决方案，其中水印技术能够实现所有权验证。然而，现有水印方法在重训练攻击下根本失效：攻击者在带水印数据集上重新训练生成模型，再生成的高效用数据不再携带水印。我们通过引入放射性（即水印在生成模型重训练后仍可检测的性质）来应对这一挑战，并提出RaMark——一种将正弦依赖关系嵌入数据分布固有成分的放射性水印方法。通过将水印与底层分布耦合，RaMark确保任何保持数据效用的生成模型也必须保留水印。我们理论上证明，去除水印会以高概率降低效用并改变数据分布。在两个真实表格数据集上，在包含10^5个独立数据所有者的大规模所有权验证设置下进行的大量实验表明，RaMark比七种最先进方法实现了显著更强的放射性，并且在面对重训练攻击和数据修改攻击时始终优于它们。

## Abstract
Recent advances in generative modeling have made generated tabular data a practical solution for privacy-sensitive data sharing, where watermarking enables ownership verification. However, existing watermarking methods fundamentally fail under retraining attacks, in which an adversary retrains a generative model on a watermarked dataset and regenerates high-utility data that no longer carries the watermark. We address this challenge by introducing radioactivity, the property that a watermark remains detectable after generative model retraining, and propose RaMark, a radioactive watermarking method that embeds a sinusoidal dependency as an intrinsic component of the data distribution. By coupling the watermark with the underlying distribution, RaMark ensures that any generative model preserving data utility also has to preserve the watermark. We theoretically show that with high probability removing watermark degrades utility and alters data distribution. Extensive experiments on two real-world tabular datasets, under a large-scale ownership verification setting with $10^5$ independent data owners, demonstrate that RaMark achieves substantially stronger radioactivity than seven state-of-the-art methods and consistently outperforms them against both retraining and data modification attacks.

- **核心问题**：生成式表格数据在共享和交易中需要保护所有权，但现有水印方法在面对**重训练攻击**时完全失效。攻击者可以在带水印数据集上重新训练一个生成模型，然后从中采样出**高效用但无水印**的新数据，从而移除水印。
- **背景**：生成式建模（如扩散模型）使得高质量合成表格数据成为可能，广泛应用于医疗、金融等隐私敏感领域。数据共享和交易需要可追溯的所有权验证（水印）。然而，现有方法（无论是基于数据库水印还是生成式水印）嵌入的信号要么是记录级的扰动、要么是生成时的启发式规则，这些信号都不是数据分布的固有成分，因而重训练的生成模型在学习分布时不会复现它们。
- **整体意义**：论文首次在生成表格数据领域引入**放射性**（radioactivity）概念——即水印在生成模型重训练后仍可检测的性质。并提出RaMark方法，通过将水印作为数据分布的内在成分（正弦依赖）嵌入，使得**保留数据效用必然保留水印**，从根本上对抗重训练攻击。

## 2. 论文提出的方法论

### 核心思想
- 将水印定义为**在特定秘密二维投影空间中指定频率的正弦依赖关系**。该依赖被嵌入到生成数据的分布中，成为数据固有的统计结构。任何重训练的生成模型，若要逼近原数据分布（从而保持效用），就必须同时学习并复现该正弦依赖，从而使水印“放射性”地保留在再生数据中。
- 核心原理：**效用保留 → 分布接近 → 谱功率接近 → 水印可检测**。理论分析证明（定理4.1和4.2），只要攻击者不显著改变数据分布（即保持效用），则水印对应的谱功率就会保持高水平。

### 关键技术细节

1. **水印检测（Detection）**：
   - 使用秘密密钥 \(\mathcal{K} = \{e_u, e_v, \beta, s, \omega\}\)，将数据集 \(Q\) 映射到二维投影平面，得到点 \((u_i, v_i)\)。
   - 对 \(u\) 坐标进行分箱（binning），计算每个箱内 \(v\) 的均值，得到离散时间信号 \(S\)。
   - 使用**Lomb–Scargle周期图（LSP）**计算信号在指定频率 \(\omega\) 处的谱功率 \(L(\omega)\)，并基于假警概率（FAP）计算检测分数 \(DS(\omega)=1-\text{FAP}(\omega)\)。高 \(DS\) 表示水印存在。

2. **水印嵌入（Embedding）**：
   - 基于预训练的扩散模型（如TabDDPM），采用**水印引导的扩散采样**（watermark-guided diffusion sampling）。
   - 定义水印似然：\(\Pr(W|z_t)=\exp(-|v_t - \sin(2\pi\omega u_t)|)\)，衡量解码后的样本与目标正弦曲线的偏差。
   - 在反向去噪的每一步，将条件分布近似为高斯分布，均值向水印似然梯度方向移动：\(\mu \leftarrow \mu + \alpha \Sigma \cdot g_z\)，其中 \(g_z\) 是 \(\log \Pr(W|z_t)\) 在 \(\mu\) 处的梯度，\(\alpha\) 为引导强度。
   - 重复所有时间步，最终生成的样本自然呈现目标正弦依赖。

3. **理论分析**：
   - 定理3.1：给出了水印引导的近似条件分布。
   - 定理4.1 & 4.2：建立了**分布相似性 ↔ 信号分布相似性 ↔ 谱功率相似性**的连锁关系，证明只要保持分布接近（高效用），谱功率差异被高概率上界约束，水印得以保留。
